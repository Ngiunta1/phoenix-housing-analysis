# Phoenix Housing Market Analysis

End-to-end data analysis pipeline analyzing real estate trends using Python, SQL, and Power BI.

## Data Source

**Dataset:** [USA Real Estate Dataset](https://www.kaggle.com/datasets/ahmedshahriarsakib/usa-real-estate-dataset)

- 2.2M+ property listings across the United States
- Updated March 2024
- Filtered to Arizona/Phoenix area for analysis

_Note: Raw dataset (~175MB) not included in repository due to size. Download from Kaggle link above and place in `data/raw/` as `usa_real_estate_raw.csv`_
